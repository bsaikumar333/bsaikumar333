🚀 QUICK START:

1. Install dependencies:
   npm install react lucide-react tailwindcss postcss autoprefixer

2. Copy files to your React project:
   • GitHubAnalytics.jsx → src/components/
   • tailwind.config.js → project root
   • globals.css → src/
   • postcss.config.js → project root

3. Import in your app:
   import GitHubAnalyticsDashboard from './components/GitHubAnalytics';

4. Use in component:
   <GitHubAnalyticsDashboard />

5. Customize metrics:
   Edit values in stats array in GitHubAnalytics.jsx

6. Run your project:
   npm start (CRA) or npm run dev (Vite)

🎯 CUSTOMIZATION OPTIONS:

📌 Update Metrics
   Change values in the stats array to match your real GitHub stats

🎨 Change Colors
   Use Tailwind gradient combinations:
   • from-orange-500 to-red-400
   • from-pink-500 to-purple-400
   • from-green-500 to-emerald-400

⏱️ Adjust Animation Speed
   Modify duration prop: duration={3000} for 3 seconds

🔄 Update Streak Ring
   Change streak value: <ProgressRing streak={5} />

➕ Add More Cards
   Add items to stats array with new icon and values

📐 Change Layout
   Modify grid columns: lg:grid-cols-4 → lg:grid-cols-5
